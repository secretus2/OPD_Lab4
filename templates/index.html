📖 Справка:

  • sin(x), cos(x), tan(x), cot(x) — принимают значения в радианах или градусах
  • cot(x) = 1/tan(x) = cos(x)/sin(x) — не определен при sin(x)=0
  • arcsin(x), arccos(x) — определены только для x ∈ [-1, 1]
  • arctan(x), arccot(x) — определены для любых x
  • Точность — количество знаков после запятой (от 0 до 10)

📊 Примеры:

  • sin(30°) = 0.5
  • cos(60°) = 0.5
  • tan(45°) = 1
  • cot(45°) = 1
  • arcsin(0.5) = 30°
  • arccot(1) = 45°
